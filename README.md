# Curso do Fábio Vedoveli (Webservice RESTful com Node.js, Restify e MySQL)
[Playlist no Youtube](https://www.youtube.com/playlist?list=PLFJmwzuHdBRTBbkyH0gATtDhj6ikOIkMy)

# Passos
* Clonar o projeto
```bash
git clone https://github.com/KsmKayk/Cursovedoveli.git
```
* Acessar a pasta
```bash
cd Cursovedoveli
```
* Instalar as dependencias
```bash
npm install
```
* Criar o arquivo de configuração do banco de dados (config/db.js) a partir do arquivo de exemplo (config/db.example.js)
```bash
cp config/db.example.js config/db.js
```
